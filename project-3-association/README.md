## Deskripsi
Project ini bertujuan untuk mengelompokkan pelanggan berdasarkan
pola belanja menggunakan metode klastering.
Klastering dilakukan untuk mengetahui segmentasi pelanggan
berdasarkan karakteristik data transaksi.

## Metode Data Mining
Klastering (Unsupervised Learning)

## Algoritma
K-Means

## Dataset
Dataset berisi data pelanggan dengan atribut:
- Total Belanja
- Frekuensi Belanja
- Rata-rata Transaksi

## Tools yang Digunakan
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Proses
1. Memuat dataset pelanggan
2. Memilih atribut numerik sebagai fitur
3. Melakukan klastering menggunakan algoritma K-Means
4. Menentukan jumlah klaster
5. Visualisasi hasil klastering

## Output
- Data pelanggan dengan label klaster
- Visualisasi hasil klastering pelanggan
